Implementation of E-Procurement Software for Lagos State Public Procurement Agency

Project Summary

The Implementation of E-Procurement Software for Lagos State Public Procurement Agency project is an ongoing initiative that began in June 2019. The project aims to streamline the procurement process for public entities in Lagos State by implementing a Saas Solution software that provides an online platform for suppliers to submit bids, automate the contract award process and facilitate bid evaluation. The project also involves training procurement personnel to effectively use the software and ensure compliance with World bank regulations. The implementation of the E-Procurement software aims to increase transparency, efficiency, cost savings, promote fair competition, and reduce fraud and corruption.

Project Objective

  • To streamline and automate the procurement process, making it more transparent, efficient, and cost-effective
  • To promote fair competition and reduce fraud and corruption in procurement
  • To increase transparency in procurement processes
  • To train procurement personnel to use the E-Procurement software effectively and ensure compliance with World bank regulations

Project Scope

The Implementation of E-Procurement Software for Lagos State Public Procurement Agency project includes the following activities:

  • Identifying the procurement processes that need to be automated
  • Analyzing the current procurement processes and identifying areas for improvement
  • Selecting an appropriate E-Procurement software solution.
  • Customizing the software to meet the specific needs of Lagos State Public Procurement Agency
  • Integrating the software with existing systems
  • Testing the software to ensure that it meets the required quality standards
  • Training stakeholders on how to use the new software
  • Implementing the new E-Procurement software solution
